iRail provides free, open access to real-time information about Belgian public transport — train schedules, connections, liveboards and disruptions from NMBS/SNCB. It is one of Open Knowledge Belgium's longest-running projects and the backbone of a wide ecosystem of travel apps, research and tools.

The open-source API serves well over a million requests a day for developers, researchers and travellers across Belgium — including through the popular pyRail library. Open Knowledge Belgium hosts and maintains the iRail infrastructure so that anyone can build on open mobility data.

Explore the API documentation at [docs.irail.be](https://docs.irail.be/) and the source code on [GitHub](https://github.com/iRail/iRail).
